# 연산자


## 비교 연산자

### -eq
- 같음
```
if [ "$a" -eq "$b" ]
```

### -ne
- 같지 않음
```
if [ "$a" -ne "$b" ]
```


### -gt
- 더 큼

```
if ["$a" -gt "$b" ]
```

###  -ge
- 더 크거나 같음

```
if [ "$a" -ge "$b" ]
```

### -lt
- 더 작음

```
if [ "$a" -lt "$b" ]
```

### -le
- 더 작거나 같음
```
if [ "$a" -le "$b" ]
```

### -z
- 문자열이 "null"임. 즉, 길이가 0

### -n
- 문자열이 "null"이 아님.
